# RNinT
> read noval in your terminal.

又双一个终端看小说工具，由go语言驱动。

### 安装
1.二进制安装
（1）https://github.com/howie6879/owllook_api/releases
直接在Realease下载你需要的版本

2.自己构建
```sh
# 建议安装govendor管理依赖
go get -u github.com/kardianos/govendor

git clone https://github.com/vanishcode/RNinT.git
cd RNinT
govendor sync
go run main.go
```
